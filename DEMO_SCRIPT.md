# NutriLens — Demo Script (90 seconds)

**Opening (15s):** "We make 200+ food decisions a day. Most apps just 
count calories after. NutriLens intercepts the decision BEFORE you eat."

**Demo (50s):**
1. Open app on phone / show on screen
2. "I'm about to eat this burger." → snap/upload burger photo
3. Red scorecard appears → point to healthScore 2/10, read the "why"
4. "But look — here are 3 healthier spots within 500m."
5. Tap green pin → InfoWindow with restaurant details
6. Switch to salad → green card → streak banner appears

**Close (25s):** "Built with Gemini 2.5 Flash for vision intelligence, 
Google Maps Places API for geo-context, and deployed on Cloud Run + 
Firebase. This isn't a calorie counter — it's a behaviour change tool 
that meets you in the moment of decision."
